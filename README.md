# pinboard-offline-checker

Simple tool to mark dead links in your pinboard.in collection.

## Usage

1. Install leiningen

2. ```cd``` into project directory

3. ```lein run USER TOKEN```

You can find your token in the 'Settings' page of pinboard.in
